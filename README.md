🍔 Food Ordering System - Frontend

**📌 Giới thiệu**

Đây là frontend của hệ thống đặt đồ ăn, được xây dựng bằng React + TypeScript + Ant Design.
Hệ thống hỗ trợ:

- Quản lý món ăn (CRUD)
- Quản lý đơn hàng
- Tìm kiếm & lọc dữ liệu
- Thanh toán online (VNPAY - tích hợp backend)

**🛠️ Công nghệ sử dụng**

⚛️ React + TypeScript  
🎨 Ant Design (UI Library)  
🔄 Axios (call API)  
📦 React Hooks (useState, useEffect)

**🧩 Custom Components:**
- TableUI
- TableToolbar

**🚀 Cài đặt & chạy project**

1. Clone project  
git clone <your-repo>  
cd frontend  

2. Cài dependencies  
npm install  

3. Chạy project  
npm run dev
